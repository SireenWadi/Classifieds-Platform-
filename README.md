## Project Overview
This project is a simplified classified ads platform inspired by KupujemProdajem.com.  
Users can browse ads, search by title or description, view categories, post ads, and contact sellers.

The focus of this project is functionality, clean structure, and maintainable backend logic rather than complex UI design.

---

## Features Implemented

### Mandatory Features
- Create a new advertisement
- List advertisements in a clear grid layout
- Categories system
- Search ads by title and description

### Additional Features
- User registration and login
- Single ad detail view
- Image upload for ads
- Contact Us form with backend handling
- Seller messaging system
- Responsive layout using Bootstrap
